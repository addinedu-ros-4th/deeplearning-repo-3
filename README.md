# deeplearning-repo-3
딥러닝 프로젝트 3조. 이미지 분석을 통하여 사람의 특징을 추출하고 예측하는 딥러닝

-------------------------------------
## 역할
* 개별 역할
* 김영환 - 나이 인식 기능, 유사도분석 기능,QT,라벨링
* 김진홍 - 옷차림 인식 기능, 통신, DB,라벨링
* 유재상 - 사람 검출기능 ,체형 분석 기능,QT,라벨링
